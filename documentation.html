Документация к SDK для разработки стратегий

Турнирная система поддерживает только языки программирования C++, Pascal, Python. SDK поддерживает любые языки, но рекомендуется использовать поддерживаемые.

Системные требования:
  • ОС Windows XP или новее, Linux 2.6 или новее, Mac OS X 10.5 или новее
  • Python 3.0 или новее (рекомендуется Python 3.3)
  • Для визуализации процесса игры необходима библиотека colorama (включена в дистрибутив, устанавливается отдельно)
Установка
  1. Загрузите SDK с официальной страницы турнира.
  2. Распакуйте архив в пустую папку
  3. Откройте папку, найдите папку colorama
  4. Откройте терминал, перейдите в папку colorama и наберите python3 setup.py install
Конфигурация
  1. Скопируйте в папку bots ваши стратегии
  2. Создайте в папке с SDK пустой текстовый файл
  3. Откройте его и вставьте туда команды для запуска вашей стратегии
    4. Примеры команд для запуска:
    bots/game.exe # Для компилируемых языков
    python3 bots/game.py # Только для Python
Использование
  1. Откройте терминал, перейдите в папку с распакованным SDK
  2. Для запуска БЕЗ визуализации наберите python3 run_tools.py -c BOT_COMMAND_FILE -d pepelac, где BOT_COMMAND_FILE - файл с командами для запуска стратегий (подробнее см. "Конфигурация")
  3. Для запуска С визуализацией наберите python3 run_tools.py -c BOT_COMMAND_FILE -d pepelac -v
